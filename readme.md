## 十年oi一场空，不开`long long` 见祖宗
- `cin.peek`用于读取下一个输出字符，但是不清除该读入字符。
- `setw`用于控制输出字符宽度，不足的部分用`setfill`函数传入字符补齐。
- `queue`用于定义一个队列，语法结构为`queue <Data Type> name`,`name.pop()`用于出队，`name.front()`用于查看第一个元素.
